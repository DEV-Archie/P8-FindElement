# Find Elements Which Are Present In First Array And Not In Second
<h3><i>Given two arrays, the task is that we find numbers which are present in first array, but not present in the second array.</i><br></h3>


Input:

```java
a[] = {1, 2, 3, 4, 5, 10};
b[] = {2, 3, 1, 0, 5};
```

Output:

```java
Output: 4, 10   
```

Time Complexity:

```python
O(n^2)
```
